# 🌾 Agri Market Portal

Agri Market Portal is an Angular-based web application that provides real-time mandi (market) data, helping users track agricultural commodity prices efficiently.

---

## 🚀 Features

- 📊 Live mandi price data (API integration)
- 🔍 Search and filter products
- 📦 Product listing and detail view
- 🌙 Light/Dark mode toggle
- ⚡ Responsive dashboard UI

---

## 🛠️ Tech Stack

- **Frontend:** Angular 21
- **Styling:** CSS / Bootstrap
- **API:** Data.gov / custom backend
- **Tools:** Git, GitHub
